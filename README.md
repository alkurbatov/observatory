# Observatory

## About
Show nice-looking sprint statistics using the Jira API and the R language.

## Requirements
* nodejs >= 10
* R language support
* Rscript available in PATH

## Usage
```bash
# Install dependencies.
$ npm install

# Draw the plots.
# The results are exported to Rplots.pdf.
# The raw results are available in *.csv files.
$ npm start
```
